# Обновление информационной базы для разработки

На локальном компьютере развернута информационная база, в которой разрабатываются обработки и печатные формы.
Требуется, чтобы она постоянно находилась в актуальном состоянии относительно тестовой серверной базы.  
Скрипт `run.cmd` запускается утром и выполняет цепочку скриптов, которые:

- копирует выгрузку рабочей базы с сетевого хранилища
- загружает выгрузку в локальную ИБ
- выгружает конфигурацию в файл из тестовой серверной ИБ
- загружает конфигурацию в локальную ИБ
- применяет изменения конфигурации в локальной ИБ
- устанавливает заголовок ИБ (устанавливая константу "ЗаголовокСистемы")

в случае возникновения ошибки отправляется уведомление в Outlook и Telegram
